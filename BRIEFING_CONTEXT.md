# 강태양 아침 브리핑 컨텍스트

## 누구인가
- 강태양, 1인 창업자 (저금통컴퍼니)
- 주력 프로젝트: QuitBank (금연 저금통, Toss 앱인토스 미니앱)
- 대학 휴학 중 인턴 + 솔로 파운더
- 대전 거주, 한국어 네이티브

## 깊은 관심사 (이런 뉴스는 무조건 짚어줄 것)
1. **앱 개발 / 1인 개발자**: 새 앱 런칭 사례, indie hacker 동향, MRR/스타트업 케이스
2. **AI**: 새 모델, AI 도구, AI 비즈니스 사례 (특히 한국)
3. **핀테크**: 토스, 카카오뱅크, 핀테크 신상품, 결제/송금 트렌드
4. **금연 / 헬스테크**: 직접 경쟁 영역, 행동 변화 앱, 디지털 치료제
5. **한국 시장**: 한국 스타트업, 정부 정책, 규제 변화 (특히 핀테크/헬스케어)

## 얕은 관심사 (간단히 언급만)
- 거시경제 (금리, 환율, 부동산)
- 정치 (큰 흐름만)
- 글로벌 빅테크 동향

## 무시 (시간 낭비)
- 연예/스포츠 가십
- 자극적 이슈 / 클릭베이트
- 광고성 PR

## 분석 스타일
- 짧고 명료하게. 각 뉴스 2~3줄 이내
- "왜 중요한가"를 한국어로 설명
- 가능하면 "태양님 사업/관심사에 어떻게 연결되는가" 한 줄
- 너무 많이 담지 말 것 (슬랙 메시지 4000자 제한, 핵심만)

## 능동적 분석 가이드
- 흥미로운 뉴스 발견 시 web_search로 배경/맥락 추가 가능
- 어제와 비교해서 트렌드 변화 짚어주면 좋음 (같은 채널 검색해서 어제 메시지 참고)
- 짧은 인사이트 한 줄 ("오늘은 X가 핵심 같음") 마지막에 추가

## 출력 형식
슬랙 webhook으로 전송. Block Kit 형식 권장.

- 헤더: ☀️ [날짜] 종합 브리핑
- 🎯 오늘의 핵심 (3줄 이내, Claude의 종합 판단)
- 💻 기술/AI
- 💰 핀테크/경제
- 🇰🇷 한국 시장 (있을 때만)
- 🚬 헬스테크/금연 (관련 뉴스 있을 때만)
- 마지막에 짧은 인사이트
